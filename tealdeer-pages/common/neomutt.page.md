# neomutt

> Terminal-based email client

- Toggle tag on current message:

`tt`

- Tag all visible messages (after limit):

`tT then .`

- Untag all visible messages:

`tu then .`

- Apply command to all tagged messages:

`; (then command)`

- Delete message permanently:

`d`

- Delete messages by pattern:

`D`

- Delete all from sender:

`F5`

- Save message to folder:

`s`

- Move message to folder (selector):

`B`

- Create task from email:

`ta`

- Quick sync current account:

`o`

- Full sync with expunge:

`O`

- Commit changes (sync mailbox):

`$`

- Switch mailbox (selector):

`z`

- Switch to unread emails:

`u`

- Change folder:

`c`

- Reply to message:

`r`

- Reply to all:

`g`

- Forward message:

`f`

- Compose new message:

`m`

- Recall postponed message:

`R`

- Edit message as new:

`e`

- Search messages:

`/`

- Limit view to pattern:

`l`

- Clear limit (show all):

`l then .`

- Show sort menu:

`s (gmail account only)`

- Next/previous message:

`j/k`

- Next unread message:

`Tab`

- Previous unread:

`Shift + Tab`

- Page down in message:

`Space`

- Page up in message:

`b`

- Go to top/bottom:

`Home/End`

- View attachments:

`v`

- Toggle headers:

`h`

- View raw message:

`Esc e`

- Flag/star message:

`F`

- Mark as read:

`Ctrl + r`

- Toggle sidebar:

`Ctrl + n`

- Navigate sidebar up/down:

`Ctrl + p/Ctrl + n`

- Open folder from sidebar:

`Ctrl + o`

- Collapse/uncollapse thread:

`Space (on thread)`

- Collapse all threads:

`Esc v`

- Pattern: from sender:

`~f <pattern>`

- Pattern: subject:

`~s <pattern>`

- Pattern: body content:

`~b <pattern>`

- Pattern: any header:

`~h <pattern>`

- Pattern: unread messages:

`~U`

- Pattern: tagged messages:

`~T`

- Pattern: flagged messages:

`~F`

- Pattern: date range:

`~d <range>`

- Pattern: size:

`~z <size>`

- Pattern: has attachments:

`~X 1`

- Pattern: all visible (after limit):

`.`

- Pattern: NOT:

`!`

- Pattern: OR:

`|`

- Pattern: AND:

`space`

- Pattern: group:

`(<patterns>)`

- Pipe message to command:

`|`

- Extract URLs with urlview:

`Ctrl + u`

- Abort current action:

`Esc`

- Quit neomutt:

`q`

- Show mailbox info:

`=`

- Switch to piotrzan@gmail.com:

`i1`

- Switch to piotr.zaniewski@loft.sh:

`i2`
