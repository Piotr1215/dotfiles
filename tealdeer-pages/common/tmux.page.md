# tmux

> Terminal multiplexer shortcuts

- Split window horizontally:

`Ctrl + b %`

- Split window vertically:

`Ctrl + b "`

- Switch pane:

`Ctrl + b arrow`

- Close pane:

`Ctrl + b x`

- Create new window:

`Ctrl + b c`

- Next window:

`Ctrl + b n`

- Previous window:

`Ctrl + b p`

- List sessions:

`Ctrl + b s`

- Detach session:

`Ctrl + b d`
