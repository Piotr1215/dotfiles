# mpv

> Video player shortcuts

- Go back 5 seconds:

`Left Arrow`

- Go forward 5 seconds:

`Right Arrow`

- Go back 1 minute:

`Down Arrow`

- Go forward 1 minute:

`Up Arrow`

- Go forward 10 minutes:

`Shift + Page Up`

- Go back 10 minutes:

`Shift + Page Down`

- Go to next chapter:

`Page Up`

- Go to previous chapter:

`Page Down`

- Go to previous subtitle frame:

`Ctrl + Left Arrow`

- Go to next subtitle frame:

`Ctrl + Right Arrow`

- Toggle subtitles display:

`V`

- Select previous subtitle file:

`Shift + J`

- Select next subtitle file:

`J`

- Sync subtitle with previous frame:

`Ctrl + Shift + Left`

- Sync subtitle with next frame:

`Ctrl + Shift + Right`

- Increase subtitle delay by 100ms:

`Z`

- Decrease subtitle delay by 100ms:

`Shift + Z`

- Increase subtitle font by 10%:

`Shift + G`

- Decrease subtitle font by 10%:

`Shift + F`

- Shift subtitle placement up:

`R`

- Shift subtitle placement down:

`Shift + R`

- Switch subtitle to SSA format:

`U`

- Slow down playback by 10%:

`[`

- Speed up playback by 10%:

`]`

- Drop playback speed by 50%:

`Shift + [`

- Increase playback speed by 50%:

`Shift + ]`

- Mark position as playback loop:

`L`

- Create infinite loop between marks:

`Shift + L`

- Reset playback speed to 100%:

`Backspace`

- Decrease volume by 1:

`/ or 9`

- Increase volume by 1:

`* or 0`

- Mute/Unmute audio:

`M`

- Cycle video tracks:

`Shift + ``

- Cycle audio tracks:

`Shift + 3`

- Take screenshot:

`S`

- Take screenshot without subtitles:

`Shift + S`

- Take window screenshot:

`Ctrl + S`

- Increase audio delay by 100ms:

`Ctrl + +`

- Decrease audio delay by 100ms:

`Ctrl + -`

- Force different aspect ratio:

`Shift + A`

- Toggle deinterlacer:

`D`

- Decrease contrast:

`1`

- Increase contrast:

`2`

- Decrease brightness:

`3`

- Increase brightness:

`4`

- Decrease gamma:

`5`

- Increase gamma:

`6`

- Decrease saturation:

`7`

- Increase saturation:

`8`

- Enable hardware decoding:

`Ctrl + H`

- Increase picture cropping:

`Shift + W`

- Decrease picture cropping:

`W`

- Move picture up:

`Alt + Up Arrow`

- Move picture down:

`Alt + Down Arrow`

- Move picture left:

`Alt + Left Arrow`

- Move picture right:

`Alt + Right Arrow`

- Zoom in:

`Alt + +`

- Zoom out:

`Alt + -`

- Reset picture position and zoom:

`Alt + Backspace`

- Resize window to half size:

`Alt + 0`

- Resize window to original size:

`Alt + 1`

- Resize window to double size:

`Alt + 2`

- Play next file in playlist:

`> or Enter`

- Play previous file in playlist:

`<`

- Toggle pause:

`P or Space`

- Display next frame (frame step):

`.`

- Display previous frame:

`,`

- Toggle fullscreen:

`F`

- Exit fullscreen:

`Esc`

- Toggle always on top:

`T`

- Display progress bar:

`O or Shift + P`

- Print file info:

`I or Shift + I`

- Quit player:

`Q`

- Quit and save position:

`Shift + Q`

- Toggle pause (mouse):

`Right Click`

- Toggle fullscreen (mouse):

`Double Left Click`

- Seek forward 10 seconds:

`Scroll Up`

- Seek backward 10 seconds:

`Scroll Down`
