# vim

> Text editor shortcuts

- Enter hover window (LSP):

`K K (press K twice)`

- Copy char from line above:

`Ctrl + y (insert mode)`

- Copy char from line below:

`Ctrl + e (insert mode)`

- Surround word with parens:

`ysiw)`

- Surround to end of line with quotes:

`ys$"`

- Delete surrounding brackets:

`ds]`

- Delete surrounding HTML tags:

`dst`

- Change surrounding quotes:

`cs'"`

- Change surrounding tag type:

`csth1<CR>`

- Delete function call wrapping:

`dsf`

- Open help:

`:help`

- Save file:

`:w`

- Quit:

`:q`

- Save and quit:

`:wq`

- Force quit without saving:

`:q!`
