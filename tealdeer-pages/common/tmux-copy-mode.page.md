# tmux-copy-mode

> Tmux copy mode shortcuts (vi mode)

- Enter copy mode:

`Alt + ; (custom) or Ctrl + b [`

- Exit copy mode:

`q`

- Move left/down/up/right:

`h/j/k/l`

- Jump to next word start:

`w`

- Jump to previous word start:

`b`

- Jump to word end:

`e`

- Jump to next WORD (space-delimited):

`W`

- Jump to previous WORD:

`B`

- Jump to WORD end:

`E`

- Start of line:

`0`

- End of line:

`$`

- First non-blank character:

`^`

- Last non-blank character:

`g_`

- Go to top of buffer:

`gg`

- Go to bottom of buffer:

`G`

- Go to line number:

`:`

- Move up/down by paragraph:

`{/}`

- Jump to next occurrence of character:

`f<char>`

- Jump to previous occurrence of character:

`F<char>`

- Repeat last f/F search:

`;`

- Repeat last f/F search in opposite direction:

`,`

- Move cursor high/middle/low on screen:

`H/M/L`

- Start character selection:

`v`

- Start line selection:

`V`

- Start block/rectangle selection:

`Ctrl + v`

- Start selection (alternative):

`Space`

- Clear selection:

`Escape`

- Swap cursor to other end of selection:

`o`

- Select current word instantly:

`w (when no selection active)`

- Select current WORD:

`W (when no selection active)`

- Copy selection to clipboard:

`y`

- Copy selection and exit:

`Enter`

- Search forward:

`/`

- Search backward:

`?`

- Next search match:

`n`

- Previous search match:

`N`

- Page down:

`Ctrl + f or Page Down`

- Page up:

`Ctrl + b or Page Up`

- Half page down:

`Ctrl + d`

- Half page up:

`Ctrl + u`
