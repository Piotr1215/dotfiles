# bookmarks

> File and folder bookmarking system

- Add folder/file to bookmarks:

`addbmk -d "Description" <path>`

- Open bookmarks in tmux sessionizer:

`__sessionizer.sh (then Ctrl+L)`

- View bookmarks in file opener:

`__file_opener.sh (then Ctrl+B)`

- Edit bookmarks config file:

`nvim ~/dev/dotfiles/scripts/__bookmarks.conf`
