# CLAUDE

> Claude Code AI assistant keybindings

- View pane output in nvim (3000 lines):

`M-Y`

- Capture command output to clipboard:

`M-Z`

- Claude worktree switcher:

`M-w`

- Claude command/URL history (fzf):

`prefix + A`

- Start Claude with monitor + pane-bound kubeconfig (homelab launch default):

`claude`

- Start Claude preserving the caller/direnv launch base:

`claude_clean`

- Pair programming mode:

`claude_pair`

- Kill stuck Claude processes:

`claude-kill`

- Dry-run kill check:

`claude-check`

- Execute command and open Claude with output:

`Ctrl + G`
