# taskwarrior

> Task management shortcuts

- Add task with due date:

`task add <desc> due:tomorrow`

- Add waiting task:

`task add <desc> wait:2days`

- Add expiring task:

`task add <desc> until:2023-09-01`

- Schedule task:

`task add <desc> scheduled:next week`

- Add recurring task:

`task add <desc> due:1st recur:monthly`

- List all tasks:

`task list`

- Mark task done:

`task <id> done`

- Show overdue tasks:

`task +OVERDUE`
