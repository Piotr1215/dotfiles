# tmux-copy-toolkit

> Fast pattern-based copying (tmux-copy-toolkit plugin)

- Quickcopy - highlight patterns (URLs, paths, IPs, etc):

`Ctrl + b Q`

- Easycopy - mark start char, mark end char, copy:

`Ctrl + b S`

- Linecopy - copy full lines by label:

`Ctrl + b W`

- Quickopen - open URLs/files in browser/app:

`Ctrl + b P`

- Easymotion - jump to any character (in copy-mode):

`s`

- Easymotion 2-char search (in copy-mode):

`S s`

- Easymotion jump to lines below (in copy-mode):

`S j`

- Easymotion jump to lines above (in copy-mode):

`S k`

- Next batch of matches (in quickcopy):

`n`

- Open pane in nvim for text-object yanking (yip, yiw, etc):

`Ctrl + b Y`
