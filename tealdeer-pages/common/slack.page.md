# slack

> Team chat desktop app shortcuts (Linux/Windows keys)

- Show native shortcuts overlay in Slack (authoritative, always current):

`Ctrl + /`

- Quick switcher (jump to any channel, DM, or person):

`Ctrl + K`

- Search all messages:

`Ctrl + G`

- Search the current conversation:

`Ctrl + F`

- Compose a new message:

`Ctrl + N`

- Browse direct messages:

`Ctrl + Shift + K`

- Browse channels:

`Ctrl + Shift + L`

- Open Threads view:

`Ctrl + Shift + T`

- Open Activity (mentions and reactions):

`Ctrl + Shift + M`

- Open All Unreads view:

`Ctrl + Shift + A`

- Open Later / saved items:

`Ctrl + Shift + S`

- Open conversation info pane:

`Ctrl + Shift + I`

- Toggle right sidebar:

`Ctrl + .`

- Toggle left sidebar:

`Ctrl + Shift + D`

- Open preferences:

`Ctrl + ,`

- Previous / next channel or DM in sidebar:

`Alt + Up / Alt + Down`

- Previous / next unread channel or DM:

`Alt + Shift + Up / Alt + Shift + Down`

- Go back / forward in history:

`Alt + Left / Alt + Right`

- Jump to first unread message:

`Ctrl + J`

- Mark current conversation as read:

`Esc`

- Mark all messages as read:

`Shift + Esc`

- Mark a message as unread:

`Alt + Click message`

- Switch to workspace by number:

`Ctrl + [1-9]`

- Switch to next / previous workspace:

`Ctrl + Tab / Ctrl + Shift + Tab`

- Edit your last message (empty input box):

`Up`

- New line in a message:

`Shift + Enter`

- Add emoji reaction to focused message:

`Ctrl + Shift + \`

- Bold selected text:

`Ctrl + B`

- Italicize selected text:

`Ctrl + I`

- Strikethrough selected text:

`Ctrl + Shift + X`

- Code-format selected text:

`Ctrl + Shift + C`

- Numbered list:

`Ctrl + Shift + 7`

- Bulleted list:

`Ctrl + Shift + 8`

- Set or edit your status:

`Ctrl + Shift + Y`

- Unsend / undo last action:

`Ctrl + Z`

- Start, join, leave, or end a huddle:

`Ctrl + Shift + H`

- Toggle mute in a huddle:

`Ctrl + Shift + Space`

- Upload a file:

`Ctrl + U`

- View all downloaded files:

`Ctrl + Shift + J`

- Create a new snippet:

`Ctrl + Shift + Enter`

- Toggle full screen view:

`Ctrl + Shift + F`

- Autocomplete @name, #channel, or :emoji:

`type prefix then Tab`
